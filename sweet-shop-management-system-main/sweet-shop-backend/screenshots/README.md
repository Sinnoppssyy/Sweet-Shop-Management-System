# Screenshots Directory

This directory contains application screenshots for documentation purposes.

## Required Screenshots

1. **dashboard.png** - User dashboard showing sweet cards and search
2. **login.png** - Login page interface
3. **admin-panel.png** - Admin panel for inventory management
4. **purchase.png** - Purchase flow demonstration

## Instructions

To capture screenshots:
1. Start the backend server: `.\mvnw.cmd spring-boot:run`
2. Start the frontend: `npm start`
3. Navigate to http://localhost:3000
4. Capture screenshots of each key feature
5. Save with the filenames listed above

## Note

Screenshots should be added before project submission to demonstrate the working application.
